## üîí Privacy

Homebase is built with privacy in mind:

- No analytics, tracking, or telemetry
- Most data stays on your device (settings and cached content)
- Some enabled features make network requests (weather, RSS feeds, wallpapers, and optional search suggestions)
- No remote JavaScript execution
