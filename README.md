# Homebase New Tab Dashboard

Homebase replaces your browser’s default New Tab with a fast, customizable dashboard for everyday browsing — search, weather, news, bookmarks, wallpapers, and productivity tools in one clean page.

🔗 **Install Homebase:**  
- **Firefox Add-ons:**  
  https://addons.mozilla.org/en-US/firefox/addon/homebase-new-tab-dashboard/

- **Chrome Web Store:**  
  https://chromewebstore.google.com/detail/homebase/ejfdeilhncacmmbdfmgpolpoldpllbmc

---

## ✨ Features

- Custom New Tab dashboard with a modern layout
- Multi-engine search including Google, YouTube, DuckDuckGo, Bing, Yahoo, Yandex, Amazon, Wikipedia, and more
- Optional search suggestions from your selected provider
- Bookmarks widget for fast access
- Recent history widget optional
- Weather widget powered by Open-Meteo with location search and unit selection
- News widget using RSS with multiple sources including BBC, Al Jazeera, ESPN/Cricinfo, Feedburner feeds, and more
- Wallpaper gallery with images and videos
- Local wallpaper caching for fast loading
- Drag-and-drop organization for supported sections
- Firefox Containers support optional
- Clipboard “Paste to Save” workflow only when you trigger paste
- Settings panel with local backup and restore

---

## 🔒 Privacy

Homebase is built with privacy in mind:

- No analytics, tracking, or telemetry
- Most data stays on your device, including settings and cached content
- Some enabled features make network requests, such as weather, RSS feeds, wallpapers, and optional search suggestions
- No remote JavaScript execution

See the full [Privacy Policy](./PRIVACY.md) for details.
